# 📘 Student Data Organizer (Collection Manipulator)

## 📌 Project Description

This is a simple Python program that manages student records using basic collection data types. It allows the user to add, display, and delete student information through a menu-driven system.

---

## 🎯 Objective

The main objective of this project is to understand and apply:

* String input and formatting
* Python collection data types (List, Tuple, Set, Dictionary)
* Mutability and immutability
* Use of `del` keyword

---

## ⚙️ Features

* ✅ Add Student
* ✅ Display All Students
* ✅ Delete Student
* ✅ Menu-driven interface

---

## 🧠 Concepts Used

### 1. List

* Used to store multiple student records
* Example: `students = []`

### 2. Dictionary

* Stores each student's data (name, age, subjects, etc.)

### 3. Tuple (Immutable)

* Stores fixed data like Student ID and Date of Birth
* Example: `(student_id, dob)`

### 4. Set

* Stores subjects without duplicates

### 5. del Keyword

* Used to delete a student record from the list

---

## 🔄 Program Flow

1. Display menu options
2. Take user input
3. Perform selected operation
4. Repeat until user exits

---

## ▶️ How to Run

1. Install Python
2. Copy the code into a `.py` file
3. Run using:

```
python filename.py
```
